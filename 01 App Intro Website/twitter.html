Twitter Word Cloud

  • Input : Hashtag / Trending Topic
  • Output : Map of Topic Density
  • Audience : Current Affairs Enthusiasts
  • Data : Twitter, Google Maps

People can input a trending topic and the app will pull up data on where this topic is trending the most. The eta will also include the density of the topic in a given geographical area. This data is then visualised as a World Map with glowing dots. Each dot here represents a tweet that carries the hashtag. The density of number of dots in any area on the map will show where the topic is popular and where it is not discussed as deeply.
The data will be in real time and will display data from a specified time frame only. Another addition could be to visualise past trending topics over different time frames. This would require additional input of a time frame that the user can decide.
